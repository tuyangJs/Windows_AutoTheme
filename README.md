# Windows主题自动切换程序

这是一个简单的Windows主题模式切换工具，可以让你的系统支持日间浅色和夜间深色自动切换。
<img src="screenshot\winset.png" align="center" width=500 />

它由`Rust`构建后端代码，用于执行系统操作。

前端使用：`Typescript`+`Ant Design 5`

日落和日出功能，使用了和风天气免费Api，实现了获取日落和日出数据。

## 调试
安装依赖
```
npm install
```
调试
```
npm start
```
编译
```
npm run tauri build
```
